## Краткое описание

Программа размещает фотографии участников по кругу, случайно выбирает считалку из текстового файла и запускает процесс счёта. На каждую фотографию по очереди выводятся слова считалки. Участник, на которого выпадает последнее слово, исключается из круга. После каждого исключения круг перестраивается. Когда остаётся один участник, программа объявляет победителя, отображает корону и проигрывает звук фейерверка.

В командной версии добавлена информация о команде и участниках проекта. В главном окне отображается строка с названием команды, а по кнопке **«О проекте»** открывается окно с описанием проекта и распределением ролей.

## Состав проекта

- `main.cpp` — точка входа в приложение.
- `mainwindow.h`, `mainwindow.cpp` — основное окно, логика считалки, отображение фотографий, анимации и звуки.
- `person.h`, `person.cpp` — класс участника считалки.
- `texts/rhymes.txt` — текстовые считалки.
- `photos/` — фотографии участников.
- `sounds/` — звуки выбывания и победы.
- `images/` — дополнительные изображения.

## Как запустить проект

1. Открыть файл `schitalochka.pro` в Qt Creator.
2. Выбрать комплект Qt 6 MinGW 64-bit или другой доступный Qt 6 kit.
3. Собрать проект.
4. Скопировать папки `photos`, `texts`, `sounds`, `images` в папку рядом с исполнимым файлом, если Qt Creator не видит ресурсы из папки сборки.
5. Запустить приложение.
